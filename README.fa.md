# PD-Element Local

ابزار نصب و مدیریت Matrix Synapse و Element برای پیام‌رسانی، تماس صوتی و تماس تصویری در شبکه‌های داخلی سازمانی

---

## معرفی

PD-Element Local یک ابزار نصب و مدیریت خودکار برای راه‌اندازی کامل پلتفرم Matrix در شبکه‌های داخلی (LAN) است.

این پروژه بر پایه نسخه اصلی PD-Element ایجاد شده اما برای استفاده در محیط‌های سازمانی بازطراحی شده است.

برخلاف نسخه اصلی که از Let's Encrypt و اینترنت عمومی استفاده می‌کند، این نسخه برای شبکه‌های خصوصی طراحی شده و از زیرساخت PKI داخلی استفاده می‌کند.

به صورت خودکار موارد زیر نصب و پیکربندی می‌شوند:

- Matrix Synapse
- Element Web
- Coturn
- Nginx
- Root CA
- Intermediate CA
- گواهینامه‌های داخلی TLS
