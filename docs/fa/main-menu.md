# راهنمای منوی اصلی

این سند تمام گزینه‌های موجود در منوی اصلی **PD-Element Local** را توضیح می‌دهد.

هدف این راهنما تنها توضیح عملکرد هر گزینه نیست، بلکه مشخص می‌کند هر گزینه در چه زمانی باید استفاده شود، چه پیش‌نیازهایی دارد و پس از اجرای آن چه انتظاری باید داشته باشید.

---

# 1. نصب / نصب مجدد Matrix + Element + TURN

## هدف

استقرار یا نصب مجدد کامل مجموعه سرویس‌های ارتباطی Matrix.

این گزینه سرویس‌های زیر را نصب و پیکربندی می‌کند:

- Matrix Synapse
- Element Web
- Coturn TURN Server
- Nginx
- PKI داخلی
- گواهی‌های TLS
- پیکربندی Matrix
- پیکربندی Element
- یکپارچه‌سازی TURN
- بررسی سلامت (Health Validation)

## چه زمانی باید از آن استفاده کنم؟

- اولین نصب
- نصب مجدد یک نصب آسیب‌دیده
- مهاجرت به یک سرور جدید
- بازسازی کامل محیط

## اطلاعات موردنیاز

نصب‌کننده اطلاعات زیر را از شما درخواست خواهد کرد:

- دامنه Matrix Homeserver
- دامنه Element
- دامنه اصلی (Base Domain)
- آدرس IP شبکه داخلی (LAN) سرور
- مسیر دایرکتوری PKI
- مدت اعتبار Root CA
- مدت اعتبار گواهی سرور
- SANهای اضافی (اختیاری)

## نکات

اجرای این عملیات چندین بار ایمن است.

در صورت امکان، تنظیمات موجود مجدداً استفاده خواهند شد.

---

# 2. ایجاد کاربر مدیر (Create Admin User)

## هدف

ایجاد یک مدیر (Administrator) جدید برای Matrix.

این حساب دارای دسترسی مدیریتی خواهد بود و می‌تواند سرور را از طریق ابزارهای مدیریتی Matrix مدیریت کند.

## چه زمانی استفاده شود

- راه‌اندازی اولیه سرور
- ایجاد مدیران جدید

## توصیه

فقط کاربران مورد اعتماد باید دسترسی Administrator دریافت کنند.

---

# 3. ایجاد کاربر عادی (Create Normal User)

## هدف

ایجاد یک کاربر استاندارد Matrix.

این حساب هیچ مجوز مدیریتی ندارد.

## کاربرد معمول

از این گزینه هنگام ایجاد حساب برای کارکنان یا اعضای سازمان استفاده کنید.

---

# 4. ایجاد کاربر با رمز عبور تصادفی

## هدف

تولید خودکار یک رمز عبور قوی.

اسکریپت:

- یک رمز عبور تصادفی و امن تولید می‌کند.
- کاربر را ایجاد می‌کند.
- رمز عبور تولیدشده را نمایش می‌دهد.

## مناسب برای

- استقرارهای بزرگ
- ورود کاربران در محیط‌های Enterprise
- حساب‌های موقت

---

# 5. فعال‌سازی مجدد کاربر

## هدف

فعال کردن حساب کاربری که قبلاً غیرفعال شده است.

## کاربرد معمول

- بازگشت کارمند به سازمان
- پایان دوره تعلیق موقت

---

# 6. نمایش کاربران

## هدف

نمایش تمام کاربران Matrix که در حال حاضر روی سرور ثبت شده‌اند.

کاربردها:

- مدیریت
- ممیزی (Auditing)
- بررسی صحت ایجاد کاربران

---

# 7. غیرفعال کردن کاربر

## هدف

غیرفعال کردن یک حساب بدون حذف آن.

کاربر:

- نمی‌تواند وارد سیستم شود.
- به سرویس‌های Matrix دسترسی نخواهد داشت.

اطلاعات حساب در پایگاه داده باقی می‌ماند.

## توصیه به‌جای حذف

غیرفعال‌سازی باعث حفظ تاریخچه پیام‌ها و اطلاعات ممیزی می‌شود.

---

# 8. تنظیم محدودیت حجم بارگذاری (Set Upload Limits)

## هدف

تغییر حداکثر حجم مجاز برای بارگذاری فایل‌ها.

این گزینه تنظیمات هر دو سرویس زیر را به‌روزرسانی می‌کند:

- Nginx
- Matrix Synapse

سرویس‌ها به‌صورت خودکار مجدداً بارگذاری (Reload) خواهند شد.

## مثال

افزایش محدودیت بارگذاری از 50 مگابایت به 250 مگابایت.

---

# 9. فعال یا غیرفعال کردن ثبت‌نام

## هدف

فعال یا غیرفعال کردن ثبت‌نام عمومی کاربران.

در صورت غیرفعال بودن:

- کاربران قادر به ثبت‌نام نخواهند بود.
- مدیران همچنان می‌توانند با استفاده از این اسکریپت کاربران جدید ایجاد کنند.

## توصیه برای محیط‌های Enterprise

پس از راه‌اندازی اولیه، ثبت‌نام عمومی را غیرفعال کنید.

---

# 10. بررسی سلامت (Health Check)

## هدف

اجرای یک بررسی کامل از وضعیت سلامت سیستم.

موارد زیر بررسی می‌شوند:

- Matrix Synapse
- Element
- Coturn
- Nginx
- گواهی‌ها
- DNS داخلی
- پورت‌ها
- میزان استفاده از فضای دیسک
- میزان استفاده از حافظه

## توصیه

پس از موارد زیر این گزینه را اجرا کنید:

- نصب
- به‌روزرسانی
- تمدید گواهی‌ها
- تغییرات شبکه

---

# 11. Fix Wizard

## هدف

رفع خودکار مشکلات رایج استقرار.

نمونه‌هایی از مشکلات قابل رفع:

- پیکربندی خراب Nginx
- لینک‌های نمادین (Symbolic Links) مفقود
- سرویس‌های غیرفعال
- وضعیت نادرست سرویس‌ها

## توصیه

پیش از شروع عیب‌یابی دستی، ابتدا این ابزار را اجرا کنید.

---

# 12. تهیه نسخه پشتیبان از سرور

## هدف

ایجاد یک نسخه پشتیبان کامل.

نسخه پشتیبان شامل موارد زیر است:

- پیکربندی Matrix
- پایگاه داده
- Nginx
- Coturn
- PKI
- گواهی‌ها
- تنظیمات سرور

## توصیه

پیش از موارد زیر از سرور نسخه پشتیبان تهیه کنید:

- به‌روزرسانی
- تمدید گواهی‌ها
- تغییرات مهم در تنظیمات

---

# 13. بازیابی نسخه پشتیبان

## هدف

بازیابی یک نسخه پشتیبان که قبلاً ایجاد شده است.

این عملیات سرور را به وضعیت سالم قبلی بازمی‌گرداند.

## هشدار

بازیابی نسخه پشتیبان ممکن است تنظیمات و گواهی‌های فعلی را بازنویسی کند.

---

# 14. عیب‌یابی تماس‌ها (Call Diagnostics)

## هدف

بررسی و عیب‌یابی مشکلات تماس صوتی و تصویری.

این ابزار موارد زیر را بررسی می‌کند:

- سرور TURN
- تنظیمات TURN در Synapse
- پورت‌های موردنیاز
- پیکربندی شبکه
- آمادگی WebRTC

## توصیه

هر زمان کاربران با مشکلات زیر مواجه شدند این ابزار را اجرا کنید:

- برقراری نشدن تماس صوتی
- برقراری نشدن تماس تصویری
- مشکلات ICE Negotiation

---

# 15. به‌روزرسانی Element Web

## هدف

ارتقای Element Web به نسخه جدیدتر.

به‌روزرسانی‌کننده:

- نسخه درخواستی را دانلود می‌کند.
- تنظیمات شما را حفظ می‌کند.
- فایل‌های برنامه را جایگزین می‌کند.
- Nginx را مجدداً بارگذاری می‌کند.

## توصیه

همیشه پیش از به‌روزرسانی از سرور نسخه پشتیبان تهیه کنید.

---

# 16. حذف کامل (Full Uninstall / Purge)

## هدف

حذف کامل استقرار Matrix.

اجزای حذف‌شده شامل موارد زیر هستند:

- Matrix Synapse
- Element
- Coturn
- تنظیمات Nginx
- پایگاه داده Matrix
- فایل‌های پیکربندی

در صورت انتخاب:

- PKI محلی

## هشدار

این عملیات مخرب (Destructive) است.

فقط زمانی از آن استفاده کنید که قصد حذف دائمی سرور را دارید.

---

# 17. مدیریت PKI

## هدف

ورود به منوی مدیریت Enterprise PKI.

این منو امکانات زیر را در اختیار شما قرار می‌دهد:

- مدیریت Root CA
- مدیریت Intermediate CA
- صدور گواهی
- تمدید گواهی
- بررسی گواهی
- خروجی گرفتن از گواهی‌ها
- تهیه نسخه پشتیبان از PKI
- بازیابی PKI

مستندات کامل این بخش در فایل **pki-menu.md** ارائه شده است.

---

# 18. خروج

خروج از نصب‌کننده.

هیچ تغییری روی سیستم اعمال نخواهد شد.

---

# بهترین روش‌ها (Best Practices)

برای یک استقرار جدید، ترتیب پیشنهادی به شرح زیر است:

1. نصب مجموعه Matrix
2. اجرای Health Check
3. خروجی گرفتن از Root Certificate
4. نصب Root CA روی دستگاه‌های کاربران
5. ایجاد Administrator
6. غیرفعال کردن ثبت‌نام عمومی
7. ایجاد حساب‌های کاربران
8. تهیه نسخه پشتیبان

رعایت این ترتیب به ایجاد یک استقرار تمیز، ایمن و قابل نگهداری کمک می‌کند.
